# Data Policy

Raw public data are stored under `data/raw/` only when download access is available without credentials. Derived tables generated by RewireScope are written to `data/processed/`.

Dryad command-line downloads for Lázaro were blocked by the repository's WAF during this build. The file registry remains in `data/raw/lazaro_gomez_martinez/dryad_linkset.json`, and the dataset is kept as a yellow, pending-inspection source.

Burkle's Dryad files are mirrored on Zenodo and were downloaded from the Zenodo record. The historical Robertson network is not part of the Dryad raw files; it is a paper-figure reconstruction from a worked example and is labeled accordingly.
