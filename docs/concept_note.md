# Concept Note

RewireScope demonstrates one central claim: ecological risk becomes visible earlier when composition, topology, timing, environmental forcing, and rewiring are analyzed together.

The first viewport of the POC is not a generic biodiversity dashboard. It is a moving dependency system:

- a weekly RMBL plant-pollinator graph;
- edge turnover and rewiring decomposition;
- phenology windows that determine whether edges can execute;
- dependency concentration as a fragility signal;
- decision signals constrained by evidence provenance.

The product claim is intentionally bounded. RewireScope converts temporal ecological interaction data into structured decision signals. It does not price financial loss directly, does not claim every dataset supports full network reconstruction, and does not treat inferred association links as observed rewiring.
